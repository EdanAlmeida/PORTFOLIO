Projetos

Contratos de Trabalho C#

Programa desenvolvido usando a linguagem C# voltado para a orientação de objetos. O software registra os dados de um determinado funcionário, do departamento que ele está alocado e os contratos de trabalho. Em relação ao contrato, é possível registrar uma data que serve como filtro para buscar todos os contratos que o funcionário tem num determinado período, podendo obter também as informações de ganhos e horas trabalhadas.


Pedido de Compra C#

Programa desenvolvido usando a linguagem C# voltado para a orientação de objetos. O software registra os dados de um usuário, processa o status do pedido e adiciona itens em um carrinho. É possível verificar a quantidade de itens no pedido, bem como informações da quantidade, preço unitário e preço total do pedido.


Banco - Python

Programa desenvolvido usando a linguagem Python Voltado para orientação a objetos. O software serve para simular a criação de de uma conta bancária, transferência, adição e subtração de valores.


MathGame Python

Programa desenvolvido usando a linguagem Python voltado para orientação a objetos. O software gera cálculos de adição, subtração e multiplicação e o usuário fornece a resposta. É possível selecionar o nível do jogo e e salvar o nome do usuário bem como sua pontuação; esta também é computada. Cada acerto soma-se um ponto e, no caso de erro, subtrai-se um ponto.


Password Generator

Página desenvolvida com HTML, CSS e JS para gerar um password de forma aleatória. O usuário pode selecionar o tamanho (numero de caracteres) entre x e y. Também é possível copiar a senha gerada diretamente da página.
